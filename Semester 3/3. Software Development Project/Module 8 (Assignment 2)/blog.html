- Discuss the scope of `var`, `let`, and `const`

Ans : var type variables are accessed and reinitialized from within a function. It has no effect of scope.

let type variables solves the above mentiond issues of the var keyword, let are block scoped and can't be accessed outside of the block. but we can reassign value if we declare a variable using let.

const keyword is used to declare constant tyoe variables which value can not be reassigned and it is also block scoped and can't be accessed outside of the scope.

- What do you mean by REST API?

Ans : REST, which stands for Representational State Transfer, is an architectural style for designing networked applications. A RESTful API (Application Programming Interface) is an implementation of a web service adhering to the principles of REST.
