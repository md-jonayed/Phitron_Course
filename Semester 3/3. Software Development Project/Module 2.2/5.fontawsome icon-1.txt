we can get this class inside css file :
.test{
    padding:50;
    background-color:red;
    border:1px solid black;
    width:50%;
    text-align:center;
    margin:auto;
}
.child{
    padding:50;
    background-color:green;
    border:1px solid black;
    width:50%; it will take parent div's 50% place.
    text-align:center;
    margin:auto;
    height:10vh (means vertical height)
}
